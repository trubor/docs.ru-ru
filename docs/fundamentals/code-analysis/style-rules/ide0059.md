## <a name="overview"></a><span data-ttu-id="3caf3-116">Обзор</span><span class="sxs-lookup"><span data-stu-id="3caf3-116">Overview</span></span>

<span data-ttu-id="3caf3-117">Это правило помечает ненужные назначения значений.</span><span class="sxs-lookup"><span data-stu-id="3caf3-117">This rule flags unnecessary value assignments.</span></span> <span data-ttu-id="3caf3-118">Пример:</span><span class="sxs-lookup"><span data-stu-id="3caf3-118">For example:</span></span>

### <a name="example"></a><span data-ttu-id="3caf3-136">Пример</span><span class="sxs-lookup"><span data-stu-id="3caf3-136">Example</span></span>

```csharp
// csharp_style_unused_value_assignment_preference = discard_variable

{
    _ = wordCount.TryGetValue(searchWord, out var count);
    return count;
}

// csharp_style_unused_value_assignment_preference = unused_local_variable

{
    var unused = wordCount.TryGetValue(searchWord, out var count);
    return count;
}
```
